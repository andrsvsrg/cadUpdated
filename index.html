Pattern Singleton

Использовался для всего класса Canvas, во избежании создания второго поля для рисования

Pattern Observer

Использовался в виде слушателя за кнопками с типами фигур ( линия, круг, треугольник)

Pattern Adapter

Использовался для адаптации данных между приложением и API

Pattern Builder

Использовался для создания фигуры треугольник, при помощи класса Line(можно реализовать еще лучше)

Pattern Decorator

Использовался для создания PopUp подсказки о создании фигуры( добавлял в прототип как миксин )

Архитектура проекта
